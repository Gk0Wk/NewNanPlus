# NewNanPlus

该插件目前仅用于Minecraft牛腩小镇服务器。

欢迎来 [牛腩小镇](https://newnan.city/) 玩耍！

牛腩小镇服务器是一个高冷而有爱的MC小服，主营原版生存、建筑和红石，运行版本为Java国际版1.14.4。无论你是大佬还是小白，我们都欢迎你的到来，希望你在这里玩得开心。
